# KSTIA_Web — Korea Ski Tourism Instructors Association 홈페이지  

대한관광스키지도자연맹을 위한 공식 웹사이트로,  
연맹 소개, 회원 안내, 행사 및 공지, 스키·스케이팅 정보 제공 등을 목적으로 제작된 정적 웹 사이트입니다.

---

## 프로젝트 개요  

- **프로젝트명** : KSTIA_Web  
- **목적** : 스키/스케이팅 연맹의 정보 제공 및 홍보, 회원 및 일반 사용자 대상 안내  
- **주요 페이지**  
  - 메인 페이지 (Home / Index)  
  - About (연맹 소개)  
  - Contact (문의/연락처 안내)  
  - Skating / Ski 섹션 (종목 소개 및 정보 제공)  
  - 공지/이벤트, 굿즈 안내 등 추가 콘텐츠 제공 가능  

- **디자인 및 기술 스택**  
  - HTML5 / CSS3 / JavaScript  
  - CSS 폴더, fonts, 이미지, JS 리소스 폴더 구조로 정리되어 있음  
  - 정적 웹사이트 구조로, 별도의 백엔드 없이도 배포 가능  

---

## ✅ 주요 기능 / 특징  

- 연맹 및 활동 소개 페이지 — 연맹의 목적, 활동 내역, 소개글 제공  
- 스키 종목 정보 페이지 — 종목별 특징, 지도자 안내, 참고 정보 등  
- 연락 / 문의 페이지 — 사용자 문의를 위한 연락처 또는 안내 정보 제공  
- 반응형 혹은 기본 웹 호환 레이아웃 — HTML/CSS로 구성된 심플하면서 직관적인 UI  
- 정적 자원 기반 운영 — 서버 없이 GitHub Pages 등으로도 배포 가능  

---
## 사용 방법

1. 저장소를 클론 또는 다운로드합니다.  
   git clone https://github.com/425range/KSTIA_Web.git
2. 파일 구조를 유지한 채, index.html 파일을 브라우저로 열면 웹사이트를 확인할 수 있습니다.

3. 별도의 서버 없이도 정적 호스팅 서비스 (예: GitHub Pages, Netlify, Vercel 등)에 업로드하여 배포 가능합니다.

4. 필요 시 CSS / 이미지 / JS 등을 수정하여 디자인, 콘텐츠, 구조를 변경할 수 있습니다.


### 🔧 향후 계획 / 개선 아이디어
반응형 웹 디자인 적용 (모바일/태블릿에서도 최적화된 UI)

사용자 멤버십 / 로그인 기능 추가 (회원 전용 페이지) — 필요한 경우 백엔드 연동

공지사항 / 이벤트 관리 기능 추가 (CMS 혹은 간단한 관리 인터페이스)

지도 기반 스키장 위치 안내 기능 추가 (예: Google Maps / Leaflet 연동)

언어 지원 확장 (한국어 + 영어 등 다국어 버전)
